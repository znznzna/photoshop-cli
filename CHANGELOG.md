# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-XX

### Added
- CLI tool (`psd`) for Photoshop file operations
- MCP server (`psd-mcp`) for AI agent integration
- UXP Plugin for WebSocket bridge to Photoshop
- `psd file open/close/save/list/info` commands
- `psd system ping` command
- `psd schema` introspection command
- `psd mcp install/uninstall/status/test` commands
- JSON/text/table output formats with field filtering
- Dry-run mode for mutation operations
- Connection resilience with automatic reconnection
